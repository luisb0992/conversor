## Conversor de monedas

Este es un Conversor de monedas que convierte de cualquier moneda a cualquier otra moneda.
Directo, rápido y sencillo de usar.

### Instalación

```
composer install
php artisan migrate
npm install
npm run dev
```

### Tecnologías

- Laravel 10
- Vue 3
- Tailwindcss 3
- Inertiajs
